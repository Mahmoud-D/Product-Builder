# Product Builder
<b> React CRUD APP</b> 
## Tech Stack

- React
- TypeScript
- Tailwind CSS

## Getting Started

1. Clone the repository

```bash

git clone https://github.com/Mahmoud-D/product-builder.git
```


2. Install dependencies
```bash
   npm install
```

   3. Start the development server
```bash

   npm run dev
```
